v-hotkey ​

简介 ​

v-hotkey 指令用于监听键盘事件，并在事件发生时执行指定的回调函数，支持单个或多个按键的组合。

基本用法 ​

将 v-hotkey 指令添加到需要监听键盘事件的元素上，并指定要监听的按键组合和回调函数：

🎯 单键快捷键监听

尝试按 Space / Enter / Esc

按 Space

按 Enter

按 Esc

点击查看代码
vue
<template>
  <div class="container">
    <h2>🎯 单键快捷键监听</h2>
    <p>尝试按 <strong>Space / Enter / Esc</strong></p>
    <div class="box">
      <p v-hotkey="{ key: 'space', callback: handleSpace }">
        按 <kbd>Space</kbd>
      </p>
      <p v-hotkey="{ key: 'enter', callback: handleEnter }">
        按 <kbd>Enter</kbd>
      </p>
      <p v-hotkey="{ key: 'esc', callback: handleEsc }">按 <kbd>Esc</kbd></p>
    </div>
  </div>
</template>

<script lang="ts" setup>
import { vHotkey } from "@cp-vuedir/core";

const handleSpace = () => alert("🚀 Space 被按下！");
const handleEnter = () => alert("✅ Enter 被按下！");
const handleEsc = () => alert("❌ Esc 被按下！");
</script>

多个按键组合 ​

可以同时监听多个按键组合，当所有按键都被按下时，回调函数将被执行，按键间通过 + 符号连接：

🎯 多键组合快捷键监听

尝试按：

  • Ctrl + Enter → 提交
  • Alt + Shift + ArrowUp → 上移
  • Cmd + S / Ctrl + S → 保存
点击查看代码
vue
<template>
  <div class="container">
    <h2>🎯 多键组合快捷键监听</h2>
    <p>尝试按：</p>
    <ul>
      <li><kbd>Ctrl</kbd> + <kbd>Enter</kbd> → 提交</li>
      <li><kbd>Alt</kbd> + <kbd>Shift</kbd> + <kbd>ArrowUp</kbd> → 上移</li>
      <li>
        <kbd>Cmd</kbd> + <kbd>S</kbd> / <kbd>Ctrl</kbd> + <kbd>S</kbd> → 保存
      </li>
    </ul>
  </div>
</template>

<script lang="ts" setup>
import { vHotkey } from "@cp-vuedir/core";

const handleSubmit = () => alert("🚀 提交成功！");
const handleMoveUp = () => alert("⬆️ 元素已上移！");
const handleSave = (event: Event) => {
  event.preventDefault(); // 阻止浏览器默认保存行为
  alert("✅ 自定义保存操作！");
};
</script>

<style scoped>
.container {

  margin-top: 50px;
}
ul {
  list-style: none;
  padding: 0;
}
li {
  background: #f1f1f1;
  display: inline-block;
  padding: 10px;
  margin: 5px;
  border-radius: 6px;
}
kbd {
  background: #333;
  color: #fff;
  padding: 3px 6px;
  border-radius: 4px;
}
</style>

自动避免可编辑元素冲突 ​

当指令所在的元素是可编辑元素（如 input、textarea 等）时，会自动避免与可编辑元素的默认快捷键冲突，您可以放心使用。

🎯 可编辑元素自动处理

输入框内不会触发快捷键：

按 Enter（不在输入框时）

按 Ctrl + Enter

按 Esc

点击查看代码
vue
<template>
  <div class="container">
    <h2>🎯 可编辑元素自动处理</h2>
    <p>输入框内不会触发快捷键：</p>
    <textarea placeholder="在这里输入，Enter 不会触发"></textarea>
    <input type="text" placeholder="输入框中也不会触发快捷键" />

    <p v-hotkey="{ key: 'enter', callback: handleEnter }">
      按 <kbd>Enter</kbd>（不在输入框时）
    </p>
    <p v-hotkey="{ key: 'ctrl+enter', callback: handleCtrlEnter }">
      按 <kbd>Ctrl + Enter</kbd>
    </p>
    <p v-hotkey="{ key: 'esc', callback: handleEsc }">按 <kbd>Esc</kbd></p>
  </div>
</template>

<script lang="ts" setup>
import { vHotkey } from "@cp-vuedir/core";

const handleEnter = () => alert("🚀 Enter 被按下！（仅限非输入框）");
const handleCtrlEnter = () => alert("✅ Ctrl + Enter 被按下！");
const handleEsc = () => alert("❌ Esc 被按下！");
</script>

<style scoped>
.container {
  text-align: center;

  margin-top: 50px;
}
textarea,
input {
  display: block;
  width: 80%;
  margin: 10px auto;
  padding: 10px;
  border-radius: 5px;
  border: 1px solid #ddd;
}
kbd {
  background: #333;
  color: #fff;
  padding: 3px 6px;
  border-radius: 4px;
}
</style>

API ​

属性名说明类型是否必选默认值
event触发事件时的回掉函数Function必选无
value监听的按键组合或单个按键，多个按键间通过 `+` 符号连接String必选无
