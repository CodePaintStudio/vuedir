v-focus 指令 ​

基础用法 ​

当前输入内容:

vue
<script setup lang="ts">
import { ref } from "vue";

const message = ref("");
</script>

<template>
  <div>
    <input
      v-focus
      v-model="message"
      type="text"
      placeholder="这个输入框将自动获得焦点"
    />
    <p>当前输入内容: {{ message }}</p>
  </div>
</template>

功能说明 ​

  • 支持在元素挂载时自动聚焦
  • 支持在元素更新时自动聚焦
  • 适用于表单输入框、模态框等场景
  • 支持动态控制是否启用聚焦功能

注意事项 ​

  • 仅适用于可获得焦点的元素（如 input、textarea 等）
  • 在同一页面中同时只能有一个元素获得焦点
